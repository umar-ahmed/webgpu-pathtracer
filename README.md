## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) (v18 or higher)

### Installation

1. Clone repo
2. Run `corepack enable` to setup Yarn
3. Run `yarn install` to install dependencies
4. Run `yarn dev` to start the development server
